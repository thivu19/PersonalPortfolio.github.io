ARGO Data is developing a two directional UX master component library (MCL). This project will intersect development with design. Students will gain practical domain experience with StorybookJS in an Azure environment. The MCL will provide scalability and enforce standards documentd by the UX program. The functional components can be edited in the MCL and consumed across all projects. If a project develops an improvement for a functional component in the MCL, that improvement can be modified back to the MCL.

Abstract

Our team will support developing a bi-directional UX master component library (MCL) for Argo Data. The figure to the right illustrates how this product will intersect development with design. This opportunity will allow us to gain practical domain experience with StorybookJS in an Azure environment. The MCL will provide scalability and enforce standards documented by the UX program.The functional components will have the functionality to be edited in the MCL and consumed across all projects. If a project develops an improvement for a functional component in the MCL, that improvement can be modified back to the MCL.
